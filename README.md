# i3dots
My Manjaro i3 setup.

## Goals:
+ Simple and functional.
+ Easy on the eyes.
+ No extensive ricing.
+ A distraction free environment for my works.

## Software List:
+ Nvim (Atlas colorscheme)
+ Rofi
+ Pywall
+ Alacritty
+ Fish shell
+ Polybar [adi1090x](https://github.com/adi1090x/polybar-themes)
+ nnn file manager
